# Devang Lights - Admin Portal

A fully responsive, modern admin portal for managing warehouse inventory including lights, crafts, d√©cor items, and all stock.

## Features

- **Dashboard**: Total SKUs, low stock alerts, stock valuation, quick actions
- **Products (Catalog Management)**: Add/edit SKUs, import via Excel/CSV, search & filters
- **Inventory & Warehouse**: View stock per warehouse, add/adjust stock, transfers
- **Purchase Orders & Suppliers**: Create PO, track status, maintain supplier list
- **Media Library**: Upload product images, catalogs, PDFs
- **Reports**: Stock valuation, fast/slow-moving items, stock movement
- **User & Role Management**: Admin, Manager, Staff roles with permissions
- **Activity Log**: Track all changes with user and timestamp

## Mobile Support

The portal is fully responsive and can be embedded in React Native apps via WebView. No separate mobile development required.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- React Router
- Tailwind CSS
- Lucide React (Icons)
